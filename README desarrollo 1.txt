Desarrollo de HU1 ingreso de datos
Para realizar la conexion a la bd porfavor en el archivo .env cambiar los campos de user, password, host y port al que vaya utilizar
y para la creacion de la base de datos siga el formato del archivo de db.sql

para iniciar el programa en una terminal talque en la carpeta pernstack aplicar el comando npm run dev y para activar el cliente
front end cambiar el directorio a la carpeta y correr el npm start

las url del backend estan en localhost:4000 y los distintos metodos se encuentran en task.routes.js
las url del frontend estan en localhost:3000

por problemas de conexion el metodo put y delete no quedaron implementados
en el metodo get quedo parcialmente implementado debido a que por algun problema no devuelve todos los campos
